#  Smart Expense Tracker

A simple and interactive expense tracking web app to manage daily spending and visualise expenses with charts.

##  Features
- Add and display expenses with category & notes
- Real-time total spending updates
- Category-wise pie chart using Chart.js
- Clean UI with responsive layout

##  Tech Stack
- HTML, CSS, JavaScript
- Chart.js (Pie Chart)
- LocalStorage (to store expenses)
